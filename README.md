# Library Management System

A comprehensive **Library Management System** built primarily with C++. This project helps librarians and users manage library resources and users efficiently and securely.

## Features

- **Book Management:** Add, remove, or update book records.
- **User Management:** Register new users, delete, or update user information.
- **Borrow and Return:** Track book borrowing and returning operations.
- **Search and Listing:** Quickly search and filter among books and users.
- **Reporting:** Generate reports on borrowed books, overdue returns, and general statistics.
- **User-Friendly Interface:** Simple, easy-to-use text-based menu interface.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mrtoncl/Library-Management-System.git
   cd Library-Management-System
   ```

2. Build:
   - **Using CMake:**
     ```bash
     mkdir build
     cd build
     cmake ..
     make
     ```
   - **Using Makefile:**
     ```bash
     make
     ```

3. Run the application:
   ```bash
   ./LibraryManagementSystem
   ```

## Requirements

- A modern C++ compiler (C++11 or later)
- CMake (optional, for CMake build)
- Make
